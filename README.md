# 🌿 GlowCare – Skin & Body Care Landing Page

## 📌 Project Overview
GlowCare is a modern responsive ecommerce landing page for skin care and body care products.  
It is built using **HTML, CSS, and JavaScript** with interactive UI features like cart system, animations, filtering, and dark mode.

---

## ✨ Features

- 🛍️ Product listing (skincare & body care items)
- 🛒 Add to cart functionality (counter system)
- 🌙 Dark mode toggle
- 🔍 Search & filter products
- ✨ Smooth scroll navigation
- 🎬 Scroll animations
- 💎 Modern UI with hover effects
- 📱 Fully responsive design

---

## 🧱 Tech Stack

- HTML5 → Structure  
- CSS3 → Styling & animations  
- JavaScript (Vanilla JS) → Interactivity
